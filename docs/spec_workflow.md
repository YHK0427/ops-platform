# 섹션 A: 관리자 워크플로우 (전체 흐름)

> **핵심:** 이 플랫폼은 매주 **월~목** 반복 사이클로 운영됩니다.

## A-1. 주간 타임라인 전체 그림

```
월요일         금요일       토요일            일요일          수요일       목요일
│              │            │                 │               │            │
▼              ▼            ▼                 ▼               ▼            ▼
[세션 생성]  [PPT 마감]  [행사 당일]      [사후사유서]   [과제 마감]  [정산]
[팀 빌딩]   21:59:59    12:59 출석체크   마감 21:59:59  21:59:59    Finalize
(TEAM시)    [PPT스캔]   [PPT 지각스캔]   ← 이후 출결    [과제 스캔]  (수동수정
                        09:59:59            확정 원칙    [영상 업로드]  가능)
                        [사전불참 확인]
```

> **출결 확정 기준:** 일요일 22:00 이후 `excuse_type` 수정은 원칙적으로 마감.
> 단, FINALIZED 후에도 Admin은 비상 수동 수정 가능 — 해당 변경은 `ADJUSTMENT` 타입으로 ledger에 자동 기록됨.

## A-2. 단계별 관리자 액션 (화면 순서)

---

### 📅 STEP 1: 세션 생성 (월요일)
**이동:** Dashboard → `[Create New Session]` 버튼

**화면: Session Wizard (3단계)**

**Step 1-1. 기본 설정**
```
Week Num: [20]                    (자동 제안: 마지막 세션 +1)
Title:    [개인 발표 세션      ]  (영상 업로드 제목에 사용됨)
Date:     [2024-05-18]            (토요일 날짜)
Type:     ● INDIVIDUAL  ○ TEAM
Config:
  [✓] PPT 제출 있음
  [✓] 리뷰/과제 있음
  [ ] 공휴일 (체크 시 벌점 없음)
```
> 영상 게시글 제목 미리보기: `20주차_개인 발표 세션_{발표자명}`

**Step 1-2. 팀 구성 (Type=TEAM일 때만 표시, INDIVIDUAL이면 SKIP)**
→ **A-3 팀빌딩 플로우** 참고

**Step 1-3. 확인 & 생성**
```
생성할 세션 요약 표시
[Create Session] 버튼
→ DB: sessions INSERT, status='SETUP'
→ INDIVIDUAL: 모든 활성 멤버에 대해 assignments 자동 생성
→ TEAM: 팀 확정 후 PPT는 팀 단위, 나머지는 개인 단위 assignments 생성
```

**생성 완료 후:** 자동으로 해당 Session Manager 페이지로 이동

### 🔄 STEP 10: 수시 작업 (언제든)

**상점 부여 (세션 외):**
- Ledger 페이지 → `[Grant Merit]` → 수상자 + 사유 선택

**디파짓 재충전:**
- Ledger 페이지 → `[Manual Transaction]` → `Deposit Recharge` → 금액 입력

**멤버 추가/수정/비활성화:**
- Members 페이지 → `[Add Member]` (이름, 이메일, 태그 입력) 또는 각 멤버 행 `[Edit]` / `[Deactivate]`
- 태그는 자유롭게 추가/제거 가능. `leader` 태그 유무가 팀빌딩에서 리더 배분에 사용됨

**중간 이탈자 처리:**
- Members 페이지 → 해당 멤버 `[Deactivate]` 클릭
- 모달 표시:
  ```
  ┌────────────────────────────────────────────────┐
  │  김민준 비활성화                               │
  │  현재 디파짓: 18,000원                         │
  │  누적 벌점:   -2점                             │
  │                                                │
  │  환불 금액: 18,000원  (현재 디파짓 잔액 그대로) │
  │  [환불 처리 후 비활성화]   [취소]              │
  └────────────────────────────────────────────────┘
  ```
- 처리 결과:
  - `members.is_active = false`, `deactivated_at = now()`
  - ledger에 `DEPOSIT_REFUND` 기록 (amount_krw = +current_deposit, deposit_after = 0)
  - `members.current_deposit = 0`
  - 이후 해당 멤버는 모든 목록에서 기본 숨김 (필터 토글로 조회 가능)
- ⚠️ 이탈 시점에 진행 중인 세션이 있으면 → 해당 세션 출결/과제는 EXCUSED로 자동 처리

## A-3. 팀빌딩 플로우 (TEAM 세션 전용)

> INDIVIDUAL 세션이면 이 단계 전체 건너뜀

### 팀빌딩 진입점

Session Wizard Step 1에서 `Type = TEAM` 선택 시 자동으로 Step 2로 이동.
또는 이미 생성된 SETUP 상태 세션에서 `Prep 탭 → [팀 구성]` 버튼으로 재진입.

### 팀빌딩 화면 레이아웃

```
┌─────────────────────────────────────────────────────────────────┐
│  Week 20 팀 구성                                                 │
│  팀 수: [3 ▼]   [Auto Generate] 버튼                           │
├──────────────────┬──────────────┬──────────────┬────────────────┤
│  미배정 멤버          │   Team A       │   Team B       │   Team C       │
│                       │                │                │                │
│  [이지은][frontend]   │ [김민준][leader]│ [박준혁][leader]│ [최유나][leader]│
│  [한지민][design]     │ [정서연][기획]  │ [오세훈][backend]│ [강동원][backend]│
│                       │ [홍길동][design]│ [이민지][frontend]│ [한지민][design]│
│                  │              │              │                │
│  ← 드래그&드롭으로 멤버 이동 가능 →                             │
├──────────────────┴──────────────┴──────────────┴────────────────┤
│  충돌 경고: 김민준↔박준혁 (3회 같은 팀) ← 자동 감지            │
│                                                    [팀 확정]    │
└─────────────────────────────────────────────────────────────────┘
```

### [Auto Generate] 버튼 동작

1. `POST /sessions/{id}/teams/generate { num_teams: 3 }`
2. 백엔드에서 팀빌더 알고리즘 실행 (아래 섹션 8 참고)
3. 결과를 Kanban에 자동 배치
4. Admin이 드래그&드롭으로 수동 조정 가능

### [팀 확정] 버튼 동작

```
PATCH /sessions/{id}/teams { teams: [
  { name: "Team A", members: [1, 5, 9] },
  { name: "Team B", members: [2, 6, 10] },
  { name: "Team C", members: [3, 7, 11] }
]}
→ teams, team_members 테이블 INSERT/REPLACE
→ session status: SETUP → PREP
→ 개인 과제용 assignments 자동 생성 (리뷰, 피드백, 과제 = 전원 개인)
→ PPT assignments = 팀 단위 생성 (team_id 설정)
```

### 팀 세션 PPT 스캔 결과 표시

```
┌─────────────────────────────────────────────────────────┐
│  Team A: ✅ PASS  (김민준이 금 21:00 제출)               │
│  Team B: ❌ MISSING                                      │
│    └ 전체 벌금 or 리더만 벌금 결정:                      │
│      ● 팀원 전체 -2점/-3,000원  ○ 리더만  [확인]         │
│  Team C: ⏳ LATE  (박준혁이 토 09:50 제출)               │
└─────────────────────────────────────────────────────────┘
```
